1.String、StringBuffer、StringBuilder

三者异同：

​ String：不可变字符串

​ StringBuffer：可变字符串，线程安全，效率低

​ StringBuilder：可变的字符串序列，jdk5.0新增，线程不安全

​ 三者底层均为char[]

String分析：

​ String实例化方式：

​ 方式一：通过字面量定义的方式

​ 方是二：通过new构造器的方式

​ String s = “abc” ===> s = new char[]{‘a’,’b’,’c’}

​ String s = new String(“abc”)===》s = new char[‘a’,’b’,’c’]，共创建了两个对象，一个是在堆内存中new的对象，另一个是在方法区常量池中new的char[]对应的对象
